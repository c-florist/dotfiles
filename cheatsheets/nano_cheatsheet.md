# Nano Editor Cheatsheet

## File Operations
- `Ctrl+O` - Save file (WriteOut)
- `Ctrl+X` - Exit nano
- `Ctrl+R` - Read/Insert file

## Navigation
- Arrow keys - Move cursor
- `Ctrl+A` - Go to beginning of line
- `Ctrl+E` - Go to end of line
- `Ctrl+Y` - Page up
- `Ctrl+V` - Page down
- `Alt+\` - Go to beginning of file
- `Alt+/` - Go to end of file
- `Ctrl+_` - Go to line number

## Search & Replace
- `Ctrl+W` - Search (Where is)
- `Alt+W` - Search next occurrence
- `Ctrl+\` - Search and replace

## Editing
- `Ctrl+K` - Cut current line
- `Ctrl+U` - Paste (Uncut)
- `Ctrl+J` - Justify paragraph
- `Alt+6` - Copy current line
- `Ctrl+T` - Spell check

## Selection & Deletion
- `Alt+A` - Start/stop marking text
- `Ctrl+K` - Cut marked text (after marking)
- `Ctrl+D` - Delete character under cursor
- `Ctrl+H` - Delete character before cursor (Backspace)
- `Alt+Delete` - Delete word to the right

## Undo/Redo
- `Alt+U` - Undo
- `Alt+E` - Redo

## Other Useful Commands
- `Ctrl+G` - Display help
- `Ctrl+C` - Show current position
- `Alt+D` - Toggle line numbers
- `Alt+X` - Toggle help bar

## Tips
- Most commands show at the bottom of the screen with `^` meaning Ctrl
- Press `Ctrl+G` for built-in help
- `Alt` key may also work as `Esc` key on some systems
